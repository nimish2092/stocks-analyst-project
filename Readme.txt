* Project is deployed on heroku: https://stocks-analyst.herokuapp.com/

-> Steps to run the project on local machine
1] Make sure you have node and npm installed on the machine
2] Navigate to the project directory and run the command "npm install" -> This will install all the dependencies
3] Run the server.js file using the command "node server.js"
4] Open any browser and point it to "http://localhost:3000"
